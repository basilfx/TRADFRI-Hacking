## Pinout

* A0  - LED ON     - TP7
* A1  - BTN        - TP6
* B11 - FLASH CS
* B12 - GYRO INT
* B13 - FLASH VCC
* B14 - ACC INT    - TP8
* B15 - GYRO ON    - TP9
* C10 - I2C SDA
* C11 - I2C SCL
* D13 - FLASH MCLK
* D14 - FLASH MISO
* D15 - FLASH MOSI
* F0  - SWCLK      - TP1
* F1  - SWDIO      - TP2
* F2  - SWO        - TP3
* F3  - DBG        - TP4
*       RESET      - TP5
